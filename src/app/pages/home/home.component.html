Мини-проект “social network”

Реализовать Single page application, используя тестовое API https://jsonplaceholder.typicode.com/. По умолчанию считаем, что текущий пользователь имеет ID=1 и автоматически авторизован.

Фичи проекта:

Лента постов (posts)

Вывести список всех постов в стиле соц.сети (Сделать рандомизацию порядка при выводе):

  • Тайтл;
  • текст;
  • Общая информация об авторе;
  • Список комментариев по запросу.
  • Пагинация или автоподгрузка по постам, по 20 штук на странице (реализовать на фронт-сайде т.к REST не поддерживает пагинацию)
  • Добавление нового поста.
  • Удаление собственных постов.

Альбомы

Список моих альбомов - Название, кол-во контента, превью. Страница конкретного альбома Список контента: Превью, Название Пагинация или автоподгрузка по 20 штук. Просмотр конкретной фотографии в оригинальном размере. Пагинация или автоподгрузка по 20 штук на странице Удаление альбома Контент в ресурсе https://jsonplaceholder.typicode.com/photos не работает, поэтому вместо url и thumbnailUrl нужно использовать несколько рандомных локальных изображений.

Требования к проекту:

Использование Webpack для сборки и запуска проекта Использование NPM для установки зависимостей (все зависимости должны быть в node_modules) Рабочий роутинг для SPA с history браузера. Должны быть доступны урлы / - Главная страница. Лента постов /albums/ - Список альбомов /albums/id/ - Карточка альбома с просмотром контента Использование Bootstrap Использование ES6 Использование современного JS фреймворка. Плюсом будет использование MarionetteJS (На нем пишется наш проект.)
